# 🏛 HallSync – Smart Hall Booking System (UI/UX)

HallSync is a UI/UX prototype designed to solve scheduling conflicts in university event spaces like auditoriums and seminar halls.

In many universities, multiple departments attempt to organize events simultaneously, leading to clashes in venue availability. HallSync provides a centralized platform where users can check availability and pre-book halls efficiently.
I tried to make this one spefically for my own university.

## 🎯 Problem Statement

* Event clashes due to lack of coordination
* No centralized system for hall booking
* Manual or unstructured booking processes

---

## 💡 Solution

HallSync introduces:

* A centralized hall booking platform
* Real-time visibility of booked slots
* Easy date & time-based reservation system

---

## ✨ Features (UI Prototype)

* 📅 View available and booked halls
* 🕒 Select date and time slots
* 🏛 Choose different venues (auditorium, seminar hall, etc.)
* 🔐 Login & registration interface
* 🔄 Smooth navigation between pages
